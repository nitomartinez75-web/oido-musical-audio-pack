# Oído Musical Audio Pack

Estructura inicial del pack de audio.

Sustituye los archivos `.gitkeep` por muestras MP3 normalizadas.

Reglas básicas:
- Instrumentos melódicos: C4.mp3, D4.mp3, E4.mp3, G4.mp3, A4.mp3, C5.mp3
- Sonidos de percusión/animales/naturaleza: nombre_01.mp3, nombre_02.mp3
- MP3 recomendado: 192 kbps, mono o estéreo según fuente, pico máximo -1 dB.
